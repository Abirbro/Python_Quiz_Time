# Python Fundamentals Quiz Game

A simple, text-based quiz game built in Python to test beginners' knowledge of Python fundamentals.

## Features

- Multiple-choice questions
- Immediate feedback on answers (correct or incorrect)
- Score tracking and time calculation
- Simple and fun way to practice Python basics

## Questions Covered

- Python syntax
- Variables and data types
- Functions and conditionals
- Loops and operators
